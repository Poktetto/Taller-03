Este repositorio de Github contiene el Taller 03 de Programación Orientada a Objetos, en este Taller contiene un juego de magos con hechizos incluyendo sistemas para agregar magos y hechizos almacenados en diferentes txt, por lo tanto este juego se guarda a travez de txt.

El paquete de este juego es "Taller03", su clase main es "App.java" y una clase a destacar es que esta construido bajo la Arquitectura de software, por lo tanto su Dominio y Logica estan separados, dentro del Dominio una clase a destacar seria "Hechizo.java" debido a que es la clase "padre" de todos los demas tipos de hechizos (Fuego, Agua, etc...).
